class fsspec_reference_maker.hdf.SingleHdf5ToZarr(h5f: BinaryIO, url: str, spec=1, inline_threshold=0)[source]¶

Translate the content of one HDF5 file into Zarr metadata.

HDF5 groups become Zarr groups. HDF5 datasets become Zarr arrays. Zarr array chunks remain in the HDF5 file.

Parameters
h5ffile-like

Input HDF5 file as a binary Python file-like object (duck-typed, adhering to BinaryIO is optional)

urlstr

URI of the HDF5 file.

xarraybool, optional

Produce attributes required by the xarray package to correctly identify dimensions (HDF5 dimension scales) of a Zarr array. Default is False.

specint

The version of output to produce (see README of this repo)

inline_thresholdint

Include chunks smaller than this value directly in the output. Zero or negative to disable

translate()[source]¶

Translate content of one HDF5 file into Zarr storage format.

This method is the main entry point to execute the workflow, and returns a “reference” structure to be used with zarr/fsspec-reference-maker

No data is copied out of the HDF5 file.

:returns dict with references
